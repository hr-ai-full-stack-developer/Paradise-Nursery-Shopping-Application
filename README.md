# Paradise Nursery

Welcome to **Paradise Nursery** - Your one-stop shop for beautiful plants and garden supplies!

## Project Overview

Paradise Nursery is a modern, full-featured plant shop application built with React. Our platform allows customers to browse, search, and purchase a wide variety of plants and gardening products from the comfort of their home.

## Features

- **Product Listing**: Browse our extensive collection of plants and gardening supplies
- **Shopping Cart**: Add items to your cart and manage quantities
- **Responsive Design**: Optimized for desktop and mobile devices
- **Easy Navigation**: Intuitive user interface for seamless shopping experience

## About Paradise Nursery

Paradise Nursery is dedicated to bringing nature closer to your home. We offer high-quality plants, seeds, and gardening accessories to help you create your perfect green space.

## Technology Stack

- **Frontend**: React.js
- **Styling**: CSS3
- **State Management**: Redux
- **Build Tool**: Vite or Create React App

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm start`
4. Open your browser and navigate to `http://localhost:3000`

## Project Structure

- `src/components/` - React components
- `src/pages/` - Page components
- `src/styles/` - CSS files
- `src/redux/` - Redux slices and store configuration
